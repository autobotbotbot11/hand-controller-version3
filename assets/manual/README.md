# Manual Image Frames

This folder contains the gesture frames used by the in-app User Manual.

Keep filenames stable so designers can replace images without changing code.
Manual entries live in `hand_controller/ui/manual_content.py`.

Currently used files:

- `open-palm.png`
- `left-click.png`
- `right-click.png`
- `keyboard-toggle.png`

Planned/recommended files:

- `hand-lock.png`
- `hold-reposition.png`
- `thumb-pinky-reset.png`
- `control-toggle.png`

If an image is missing, the app shows a clean placeholder instead of crashing.
